# passgen
```bash
./passgen -help
```  
```plain
Usage of ./passgen:
  -dkLen int
    	The length for the password (default 64)
  -iter int
    	The iterations for the hash (default 27500)
  -pass string
    	A password (default "start-123")
  -salt string
    	A salt for the password if empty it's random
  -saltiness int
    	The salt length (default 16)
```
